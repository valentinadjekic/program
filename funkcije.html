Funkcije u C-u

Funkcije pomažu da organizujemo kod u manje, ponovo upotrebljive blokove.

Osnovni elementi funkcije:

  • Definisanje funkcije: ime, tip povratne vrednosti, argumenti
  • Prototipi funkcija: deklaracija pre glavnog programa
  • Scope promenljivih: lokalne i globalne promenljive

Primer funkcije:

#include <stdio.h>

int saberi(int a, int b) {
    return a + b;
}

int main() {
    int rezultat = saberi(5, 7);
    printf("Rezultat: %d\n", rezultat);
    return 0;
}

Primeri često korišćenih funkcija:

Funkcija Opis
printf() Štampa tekst ili vrednosti na ekran
scanf() Učitava vrednosti sa tastature
strlen() Vraća dužinu stringa
sqrt() Računa kvadratni koren broja (math.h)
