Kontrolne strukture u C

U C programiranju koristimo kontrolne strukture za upravljanje tokom programa. Najčešće su:

  • If-else - grana izvršavanja na osnovu uslova
  • Switch - više mogućih grana na osnovu vrednosti
  • For, While, Do-while - petlje za ponavljanje blokova koda

Primer If-else:

#include <stdio.h>

int main() {
    int broj = 10;
    if (broj > 5) {
        printf("Broj je veci od 5\n");
    } else {
        printf("Broj je manji ili jednak 5\n");
    }
    return 0;
}
