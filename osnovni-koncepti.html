Osnovni koncepti C programiranja

Uvod u promenljive, tipove podataka, operatore i strukturu C programa.

1. Promenljive

Promenljive su mesta u memoriji gde se čuvaju podaci. Svaka promenljiva ima ime i tip.

  • int - čitavi brojevi
  • float - brojevi sa decimalom
  • char - jedan karakter

2. Tipovi podataka

Tipovi određuju kakve vrednosti promenljive mogu da čuvaju.

  • int, float, double, char, _Bool
  • Tipovi mogu biti modifikovani sa unsigned, long itd.

3. Operatori

Operatori se koriste za izvođenje operacija nad podacima.

  • Aritmetički: +, -, *, /, %
  • Poređenja: ==, !=, >, <, >=, <=
  • Logički: &&, ||, !

4. Struktura C programa

Osnovna struktura C programa:

#include <stdio.h>

int main() {
    // Vaš kod ovde
    printf("Hello, World!\n");
    return 0;
}
